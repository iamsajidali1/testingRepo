Bulk Logical Change
===================

#. Step

.. include:: common_select.rst

#. Step

   Push the ``Choose`` button and load the ``data_bulk_logical_change.xlsx``
   file. The ``Download template`` button will generate a blank template
   spreadsheet.

   .. image:: _static/image_021.png

#. Step

   No action is required for this step. The command is predefined in the
   template.

   + push ``Next``

   .. image:: _static/image_022.png

#. Step

   No action is required. The configuration is automatically generated
   according to the inputs from XLSX spreadsheet.

   + push ``Configure Device``

   .. image:: _static/image_023.png

#. Step

   In the ``Show commands`` tab you can see the output of the validation
   commands before and after the change.

   .. image:: _static/image_024.png

#. Step

   .. image:: _static/image_025.png

#. Step

   This view is for displaying the summary of the whole change.
   You will get email with the same content you see.

   .. image:: _static/image_026.png
