Application Programming Interface (API)
=======================================


#. /action-templates/tdc-data

   Method: GET
   Authentication type: Basic Auth - mechId:password, attuid:password (aaf)
   Endpoint for getting data from Action for Technical Data Collection.
   
   Return data type: Array of objects
   Example: [{ ID: 1, NAME: test, DESCRIPTION: testing description },{ ID: 2, NAME: tes1, DESCRIPTION: testing1 description }]
