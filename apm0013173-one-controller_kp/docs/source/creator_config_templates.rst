Configuration Templates
=======================

#. Step

   Using a modal window you can write scripts which will generate
   a configuration. Customer ID is not editable, you need to specify name of
   the template, template type and scripts or you can upload scripts.

   Type of the template can be:

   * CLI
   * JSON
   * XML

   .. image:: _static/image_030.png

#. Step

   After pressing the ``Create`` button in the modal window, you will create
   a new template with name you specified and for selected customer.

   .. image:: _static/image_031.png
