UX Designer
===========

#. Properties

   Fill the required properties for the template such as:

   * template name
   * carche template name
   * Customer
   * Description (of the template)
   * Service flag
   * Service workflow

   Please notice a cArche template needs to be created in
   ``Configuration templates``. Pressing the ``Delete`` button deletes the
   template, pressing the ``Save`` button will create a new template.

   .. image:: _static/image_027.png

#. Form

   In this step you will need to design form for ``Data collection``.
   Please use drag and drop from the left menu.

   .. image:: _static/image_028.png

#. Validation

   On this tab a user needs to define validation commands which will be
   executed on a device.
